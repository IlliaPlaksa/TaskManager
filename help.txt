Commands:
    [help] - Show all available commands
    [exit] - Exit program
    [add] - Add new Task
    [edit] - Edit existing Task
    [complete] - Complete existing Task
    [delete] - Delete Task
    [show] - Show Tasks
Input formats:
    [Title] - must be non-empty
    [ID] - must be non-negative integer
    [Priority] - available values: "high", "medium", "low"
    [Labels] - optional (can be empty). To add multiply labels write them into one line delimited with spaces "_"
    [Date] - date format = DD.MM.YYYY
    [Confirm] - must be one of the following: "y", "n"
    [File name] - must be non-empty: file_name.*
